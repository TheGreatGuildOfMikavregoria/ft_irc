# 🌐 Webserv  

A lightweight, high-performance HTTP server built from scratch in C++ by **Misha**, **Kavini**, and **Grego**.  
Our mission: to deeply understand how web servers work — from sockets to HTTP parsing — and craft a minimal yet robust alternative to nginx-style servers.  

---

## ✨ Features
- [x] Multiple client connections  
- [x] Basic HTTP methods (`GET`, `POST`, `DELETE`)  
- [x] Custom configuration files  
- [ ] CGI execution (planned)  
- [ ] Chunked transfer encoding (planned)
